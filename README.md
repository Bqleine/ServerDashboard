# ServerDashboard
ServerDashboard is a pocketmine server monitoring tool. All the infos can be found on https://serverdashboard.me
## Hosting yourself
In order to obey with the [poggit rules](https://poggit.pmmp.io/rules.edit) (especially rules A5a and A5d), the api and website are available to host yourself [here](https://github.com/Bqleine/ServerDashboardApi)
Note: you can have the patron features for free by hosting the api
